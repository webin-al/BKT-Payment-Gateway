# BKT Payment Gateway

BKT Payment Gateway is a plugin which extends the payment methods in Woocommerce for Albanian stores
Enables payments via BKT Bank API.

## Installation

Upload zip file using Wordpress Dashboard or directly to wp-content/plugins and extract.

## Contributing
This plugin is developed by [web:in development house](https://webin.al)

## Extensions
This plugin uses MPDF library to generate PDF receipt, required by BKT Bank. Check [MPDF](https://mpdf.github.io/about-mpdf/requirements-v7.html) for env requirements.
